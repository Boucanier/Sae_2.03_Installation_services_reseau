Introduction

  Le but de cette SAÉ est de configurer un serveur LAMP. Un serveur LAMP est un serveur de site Web qui utilise Linux, Apache, MySQL et PHP. Cependant, ces lettres peuvent parfois avoir d'autres significations (par exemple Linux Apache MariaDB Python). Le système LAMP est un système pratique, économique et sûr car tous les logiciels qui le composent sont des logiciels libres. Le système LAMP est utilisé par de nombreux sites comme Wikipedia, Facebook ou encore Yahoo.

Pour notre projet, nous utilisons CentOS, Apache (httpd), MySQL et PHP. CentOS est une distribution de Linux installée sur les ordinateurs de l'IUT. Apache est un serveur web qui permet de répondre à des requêtes http. MySQL est un système de base de données SQL développé par Oracle. Enfin, PHP est un langage de script utilisé sur des serveurs web. PHP signifie "PHP : Hypertext Processor". Par exemple, c'est grâce à ce langage que nous pourrons utiliser des données saisies sur une page dans une autre page, il permet de lier des données et ainsi de créer des sites web dynamiques.
